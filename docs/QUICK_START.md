## ⚡ Step 1: Get the Skill (1 minute)

### Option A: Copy-Paste (Easiest)
1. Go to [skill/SKILL.md](../skill/SKILL.md) in the repository
2. Copy all the content
3. Open Claude.ai
4. Create a new conversation
5. Paste the content into the first message

### Option B: Download
1. Download [SKILL.md](../skill/SKILL.md)
2. Open it in any text editor
3. Copy the content
4. Paste into Claude

### Option C: Use Locally
1. Install using [Installation Guide](INSTALLATION.md)
2. Access from `~/.claude-skills/power-platform/`
3. Reference in Claude conversations

## 💡 Quick Tips

### Tip 1: Be Specific
**Good:** "I need to deploy a solution from Dev to Test"
**Bad:** "How do I deploy?"

### Tip 2: Share Your Context
**Good:** "I'm an admin deploying to production"
**Bad:** "What do I do?"

### Tip 3: Save Useful Responses
- Copy checklists for future use
- Save templates for customization
- Bookmark helpful answers

### Tip 4: Use QUICK_REFERENCE.md
- Find your task in one-page lookup
- Get pointed to right reference doc
- Takes 30 seconds
